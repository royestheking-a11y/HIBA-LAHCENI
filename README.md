## 🎨 Design & Development

This premium, editorial-style personal portfolio was architected and built to reflect a high-end luxury fashion and beauty brand identity. 

**Designed and Developed by [Aurangzeb sunny](https://www.linkedin.com/in/aurangzebsunny/)**

- **Tech Stack**: Built with modern React and Vite.
- **Animations**: Driven by Framer Motion for a fluid, editorial experience.
- **Styling**: Tailwind CSS & Vanilla CSS for minimal, sophisticated layouts.
- **Bilingual Support**: Instant toggle between English and French with seamless UI transitions.

## 🚀 Running the Project Locally

1. Install dependencies:
   ```bash
   npm i
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.
